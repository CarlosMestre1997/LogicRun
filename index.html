What is Logic Run?

Logic Run is a browser-based logic game created by StartSchool. Players control Startie by writing simple instructions — not by pressing buttons. The goal is not to learn syntax, but to learn how to think.

How It Works

No Direct Control

You never move Startie yourself. You write instructions and observe the result.

Logic Over Syntax

Commands are abstract. Concepts appear naturally through play.

Learn by Failing

Every failure shows you exactly what your logic caused.
